# Deploy to Vercel

## Status

- **Git repo**: Initialized and pushed to GitHub
- **GitHub**: https://github.com/NoahWolk1/liars-poker-probability
- **Vercel**: Project linked locally

## Connect Git to Vercel (for auto-deploys)

1. Go to [vercel.com/dashboard](https://vercel.com/dashboard)
2. Open the **liars-poker-probability** project
3. **Settings** → **Git** → **Connect Git Repository**
4. Select **GitHub** and choose `NoahWolk1/liars-poker-probability`
5. Future pushes to `main` will auto-deploy
